# tmux コマンドリファレンス

## 基本情報

- **Prefixキー**: `Ctrl + s` (デフォルトの`Ctrl + b`から変更済み)
- **テーマ**: Catppuccin Latte
- **ステータスバー**: 画面上部 (左: リポジトリ名/ブランチ、右: ウィンドウ一覧)

---

## 基本コマンド

| キー         | 説明                   |
| ------------ | ---------------------- |
| `Prefix + r` | 設定ファイルをリロード |

## セッション操作

| キー              | 説明                             |
| ----------------- | -------------------------------- |
| `Prefix + d`      | セッションをデタッチ             |
| `Prefix + s`      | セッション一覧を表示・切り替え   |
| `Prefix + $`      | セッション名を変更               |
| `Prefix + (`      | 前のセッションに切り替え         |
| `Prefix + )`      | 次のセッションに切り替え         |

### コマンドラインから

```bash
tmux new -s <name>       # 新しいセッションを作成
tmux ls                  # セッション一覧
tmux a -t <name>         # セッションにアタッチ
tmux kill-session -t <name>  # セッションを削除
```

## カスタムペイン操作

### 分割

| キー          | 説明                          |
| ------------- | ----------------------------- |
| `Prefix + |`  | 縦に分割 (現在のパスを維持)   |
| `Prefix + -`  | 横に分割 (現在のパスを維持)   |
| `Prefix + \`  | 縦に分割 (フルウィンドウ幅)   |
| `Prefix + _`  | 横に分割 (フルウィンドウ高さ) |

### リサイズ

1秒以内 (`repeat-time 1000`) なら Prefix なしで連打できます。

| キー         | 説明     |
| ------------ | -------- |
| `Prefix + H` | 左に拡大 |
| `Prefix + J` | 下に拡大 |
| `Prefix + K` | 上に拡大 |
| `Prefix + L` | 右に拡大 |

### 直接移動

| キー         | 説明                                                  |
| ------------ | ----------------------------------------------------- |
| `Ctrl + h`   | 左のペインへ移動 (Vim 内なら `Ctrl + h` を転送)       |
| `Ctrl + j`   | 下のペインへ移動 (Vim 内なら `Ctrl + j` を転送)       |
| `Ctrl + k`   | 上のペインへ移動 (Vim 内なら `Ctrl + k` を転送)       |
| `Ctrl + l`   | 右のペインへ移動 (Vim 内なら `Ctrl + l` を転送)       |
| `⌘ + h/j/k/l` | ペイン移動 (Ghostty の独自シーケンス経由。Vim 内なら `C-h/j/k/l` に変換して転送) |
| `Prefix + Ctrl + l` | クリアスクリーン (`Ctrl + l` を移動に割り当てた代替) |

## Claude Code 連携

| キー            | 説明                                                                       |
| --------------- | -------------------------------------------------------------------------- |
| `Prefix + a`    | 右30%に Claude Code ペインを開く (分割済みなら右ペインへフォーカスのみ)     |
| `Prefix + D`    | 開発レイアウト: 左上=Editor / 左下=Command / 右=AI待機シェル                |
| `Prefix + A`    | ポップアップで Claude Code (ディレクトリ毎にセッション永続化、閉じても会話が残る) |
| `Shift + Enter` | マルチラインプロンプト用の改行 (CSI u シーケンスを送信)                     |

Claude Code ペイン内でも `Ctrl + h/j/k/l` はペイン移動として動作します (改行は `Shift + Enter` を使用)。

### マウスホイールの挙動

Claude Code 等の inline TUI ペインでは、ホイールで tmux のコピーモードに入り履歴を遡れます (1ティック3行)。最下段まで戻ると自動でコピーモードを抜けて再描画されます。less/vim/man 等のスクロール可能なアプリは従来どおりアプリ側に委譲されます。

## Git (lazygit)

| キー         | 説明                                                     |
| ------------ | -------------------------------------------------------- |
| `Prefix + g` | lazygit をポップアップで開く (90%サイズ、閉じれば元のペインへ) |

ステータスバー左にカレントペインのリポジトリ名 (`user/repo`) とブランチ名が常時表示されます。

## ファイルピッカー (popup)

`Ctrl + f` で fzf ベースのファイルピッカーを popup で開きます。選択したパスを**起動元ペインへ書き戻し**ます (Enter は送らないので、内容を確認してから自分で実行できます)。popup は端末中央に 90% で表示され、cwd は起動元ペインに合わせられます (lazygit popup と同じ流儀)。

| キー       | 説明                                                       |
| ---------- | ---------------------------------------------------------- |
| `Ctrl + f` | ファイルピッカーを開く (Vim 内ではページ送り `C-f` を温存) |

### popup 内の操作

| キー       | 説明                                         |
| ---------- | -------------------------------------------- |
| 入力       | ファイル名であいまい絞り込み (Files モード)  |
| `Ctrl + s` | Files (fd) ⇄ Grep (rg 全文検索) を切り替え   |
| `Tab`      | 複数選択 (マーク)                            |
| `Enter`    | 選択を確定して起動元ペインへ書き戻す         |
| `Esc`      | キャンセル (何も書き戻さない)                |

Grep モード中は `bat` で該当行をハイライトしたプレビューに切り替わります。

### 書き戻しの挙動

起動元ペインで動いているプロセスに応じて書き戻し方を変えます (判定は `pane-runs.sh` を流用)。

| 起動元ペイン                          | 書き戻し形式                          |
| ------------------------------------- | ------------------------------------- |
| AI CLI (`claude` / `codex` / `gemini`) | `@相対パス` (Claude の `@file` 参照)  |
| 通常のシェル                          | シェルエスケープ済みの相対パス        |

Grep モードの結果 (`path:line:col:text`) はパス部分のみを抽出して書き戻します。

## Vim Tmux Navigator

`Ctrl + w`をプレフィックスとして、VimとTmux間でシームレスにペイン移動できます。

| キー                    | 説明                                                |
| ----------------------- | --------------------------------------------------- |
| `Ctrl + w` → `h`        | 左のペインへ移動 (Vim内の場合はVimのウィンドウ移動) |
| `Ctrl + w` → `j`        | 下のペインへ移動                                    |
| `Ctrl + w` → `k`        | 上のペインへ移動                                    |
| `Ctrl + w` → `l`        | 右のペインへ移動                                    |
| `Ctrl + w` → `H`        | Vimにそのまま`C-w H`を送信                          |
| `Ctrl + w` → `J`        | Vimにそのまま`C-w J`を送信                          |
| `Ctrl + w` → `K`        | Vimにそのまま`C-w K`を送信                          |
| `Ctrl + w` → `L`        | Vimにそのまま`C-w L`を送信                          |
| `Ctrl + w` → `=`        | Vimにそのまま`C-w =`を送信                          |
| `Ctrl + w` → `Ctrl + w` | 純粋な`Ctrl + w`を送信                              |

### コピーモード中のペイン移動

| キー       | 説明         |
| ---------- | ------------ |
| `Ctrl + h` | 左のペインへ |
| `Ctrl + j` | 下のペインへ |
| `Ctrl + k` | 上のペインへ |
| `Ctrl + l` | 右のペインへ |

## プラグイン

### 1. TPM (Tmux Plugin Manager)

プラグインの管理を行います。

| キー               | 説明                     |
| ------------------ | ------------------------ |
| `Prefix + I`       | プラグインをインストール |
| `Prefix + U`       | プラグインを更新         |
| `Prefix + Alt + u` | 未使用プラグインを削除   |

---

### 2. tmux-yank

システムクリップボードへのコピー機能を提供します。

**設定**:

- `@yank_selection 'clipboard'`: `y`でコピー時、システムクリップボードに入る
- `@yank_selection_mouse 'clipboard'`: マウス選択時もシステムクリップボードに入る

| キー               | 説明                                         |
| ------------------ | -------------------------------------------- |
| `Prefix + y`       | コマンドラインをクリップボードにコピー       |
| `Prefix + Y`       | カレントディレクトリをクリップボードにコピー |
| (コピーモード) `y` | 選択範囲をクリップボードにコピー             |
| (コピーモード) `Y` | 選択範囲をコマンドラインに貼り付け           |

---

### 3. tmux-resurrect

セッションの保存と復元を行います。

| キー                | 説明             |
| ------------------- | ---------------- |
| `Prefix + Ctrl + s` | セッションを保存 |
| `Prefix + Ctrl + r` | セッションを復元 |

**設定**: ペインの内容も保存されます (`@resurrect-capture-pane-contents 'on'`)

---

### 4. tmux-continuum

tmux-resurrectを自動化し、定期的にセッションを保存します。

- **自動復元**: 有効 (`@continuum-restore 'on'`)
- **自動起動**: 無効 (Ghostty未対応のため)
- **保存間隔**: デフォルト15分

---

### 5. extrakto

画面上のテキストを fuzzy finder で拾って、コピーや貼り付けに使えます。`tmux-yank` より「選択」ではなく「抽出」に強い。

| キー                  | 説明                                                  |
| --------------------- | ----------------------------------------------------- |
| `Prefix + Tab`        | extrakto を開く                                       |
| (extrakto内) `Enter`  | 選択したテキストをクリップボードにコピー              |
| (extrakto内) `Tab`    | 選択したテキストを現在のペインに挿入                  |
| (extrakto内) `Ctrl + f` | フィルタ切替 (`word` / `line` / `path` / `url` など) |
| (extrakto内) `Ctrl + o` | 選択した項目を開く                                  |
| (extrakto内) `Ctrl + e` | 選択した項目をエディタで開く                        |
