# コマンドラインショートカット

zsh（emacs キーバインド）でよく使うショートカット集。

このリポジトリ独自のキーバインドは [cheatsheet.md](./cheatsheet.md) を参照。

## カーソル移動

| キー       | 動作                       |
| ---------- | -------------------------- |
| `Ctrl+A`   | 行頭へ移動                 |
| `Ctrl+E`   | 行末へ移動                 |
| `Ctrl+B`   | 1文字戻る                  |
| `Ctrl+F`   | 1文字進む                  |
| `Alt+B`    | 1単語戻る                  |
| `Alt+F`    | 1単語進む                  |

## 削除・カット（キルリング）

`Ctrl+U` / `Ctrl+K` / `Ctrl+W` で削除した内容は **キルリング** に保存され、`Ctrl+Y` で貼り付け可能。

| キー       | 動作                                       |
| ---------- | ------------------------------------------ |
| `Ctrl+U`   | カーソルから**行頭まで**削除               |
| `Ctrl+K`   | カーソルから**行末まで**削除               |
| `Ctrl+W`   | カーソルから**直前の単語まで**削除         |
| `Alt+D`    | カーソルから**次の単語まで**削除           |
| `Ctrl+H`   | 1文字削除（Backspace と同じ）              |
| `Ctrl+D`   | カーソル位置の1文字削除（空行なら exit）   |
| `Ctrl+Y`   | キルリングの内容を貼り付け（ヤンク）       |

## 履歴

| キー       | 動作                                       |
| ---------- | ------------------------------------------ |
| `Ctrl+R`   | 履歴をファジー検索（fzf）                  |
| `Ctrl+P`   | 1つ前のコマンド                            |
| `Ctrl+N`   | 1つ後のコマンド                            |
| `Alt+.`    | 直前のコマンドの**最後の引数**を挿入       |

`Ctrl+R` の挙動はこのリポジトリでカスタマイズ済み。[cheatsheet.md](./cheatsheet.md) 参照。

## 編集・変換

| キー       | 動作                                       |
| ---------- | ------------------------------------------ |
| `Ctrl+T`   | カーソル位置の文字を1つ前と入れ替え       |
| `Alt+T`    | カーソル位置の単語を1つ前と入れ替え       |
| `Alt+U`    | 次の単語を大文字化                         |
| `Alt+L`    | 次の単語を小文字化                         |
| `Alt+C`    | 次の単語の頭文字を大文字化                 |

## 制御

| キー       | 動作                                       |
| ---------- | ------------------------------------------ |
| `Ctrl+L`   | 画面クリア                                 |
| `Ctrl+C`   | 実行中のコマンド中断 / 入力中の行を破棄    |
| `Ctrl+Z`   | フォアグラウンドプロセスを一時停止         |
| `Ctrl+G`   | 検索などのモードを抜ける                   |
| `Ctrl+_`   | undo                                       |

## tmux 連携

tmux のショートカットは [docs/tmux/COMMANDS.md](../tmux/COMMANDS.md) 参照。
prefix は `Ctrl+S`。

| キー                | 動作                                         |
| ------------------- | -------------------------------------------- |
| `Ctrl+S → Tab`      | extrakto（画面から単語・パス・URL を抽出）   |
| `Ctrl+S → [`        | コピーモードに入る（vi キーバインド）        |
| コピーモード `y`    | 選択範囲をクリップボードへ（tmux-yank）      |

## よく使う組み合わせ

### コマンドの引数だけ書き直す
```
Ctrl+A         # 行頭へ
Alt+F          # コマンド名の次の単語へ
Ctrl+K         # 行末まで削除
```

### 直前コマンドの最後の引数を再利用
```
$ ls /very/long/path/to/file.txt
$ cat <Alt+.>
$ cat /very/long/path/to/file.txt
```

### 履歴から探して一部だけ書き換え
```
Ctrl+R         # 履歴検索
（候補を選んで Enter で挿入）
Alt+B / Alt+F  # 単語単位で移動
Ctrl+W         # 不要な単語を削除
```
