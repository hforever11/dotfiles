# Vim ナビゲーション チートシート

`hjkl` 連打ではなく、効率的な移動方法を使う。

## 画面単位

| キー         | 動作                           |
| ------------ | ------------------------------ |
| `<C-u>`      | 半ページ上                     |
| `<C-d>`      | 半ページ下                     |
| `<C-f>`      | 1ページ下                      |
| `<C-b>`      | 1ページ上                      |
| `zz`         | カーソル位置を画面中央に       |
| `zt`         | カーソル位置を画面上部に       |
| `zb`         | カーソル位置を画面下部に       |

## 行内移動

| キー         | 動作                           |
| ------------ | ------------------------------ |
| `f{char}`    | 行内で文字にジャンプ           |
| `F{char}`    | 行内で逆方向に文字にジャンプ   |
| `t{char}`    | 文字の手前にジャンプ           |
| `T{char}`    | 逆方向で文字の手前にジャンプ   |
| `;`          | 直前の `f/t` を繰り返す        |
| `,`          | 直前の `f/t` を逆方向に繰り返す |
| `w` / `W`    | 次の単語先頭へ                 |
| `b` / `B`    | 前の単語先頭へ                 |
| `e` / `E`    | 単語末尾へ                     |
| `0`          | 行頭へ                         |
| `^`          | 行頭（非空白）へ               |
| `$`          | 行末へ                         |

## ファイル内ジャンプ

| キー         | 動作                           |
| ------------ | ------------------------------ |
| `/{pattern}` | 前方検索（最強の移動手段）     |
| `?{pattern}` | 後方検索                       |
| `n` / `N`    | 次/前の検索結果へ              |
| `{数字}j`    | 相対行番号を見て下にジャンプ（例: `12j`） |
| `{数字}k`    | 相対行番号を見て上にジャンプ   |
| `{`          | 前の段落へ                     |
| `}`          | 次の段落へ                     |
| `gg`         | ファイル先頭へ                 |
| `G`          | ファイル末尾へ                 |
| `{数字}G`    | 指定行へ（例: `42G`）          |
| `gd`         | 定義へジャンプ（LSP）          |
| `%`          | 対応する括弧へ                 |

## 履歴ベース

| キー         | 動作                           |
| ------------ | ------------------------------ |
| `<C-o>`      | ジャンプリストで前に戻る       |
| `<C-i>`      | ジャンプリストで次に進む       |
| `gi`         | 最後にInsertした位置に戻る     |
| `g;`         | 変更リストで前の変更位置へ     |
| `g,`         | 変更リストで次の変更位置へ     |

## マーク

| キー         | 動作                           |
| ------------ | ------------------------------ |
| `m{a-z}`     | ローカルマークを設定           |
| `'{a-z}`     | マーク位置の行頭へジャンプ     |
| `` `{a-z} `` | マーク位置の正確な位置へジャンプ |
| `''`         | 直前のジャンプ位置に戻る       |

## おすすめプラグイン

| プラグイン              | 説明                                         |
| ----------------------- | -------------------------------------------- |
| `flash.nvim`            | 画面内の任意の位置に2-3キーでジャンプ。`f/t` の上位互換 |
| `leap.nvim`             | flash.nvim と同様の高速ジャンプ              |
| `accelerated-jk.nvim`   | `j/k` の移動速度が押し続けると加速する       |
