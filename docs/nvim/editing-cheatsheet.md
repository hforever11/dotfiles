# Neovim Editing Cheatsheet

編集・移動・テキストオブジェクト系のメモをまとめたもの。

## Navigation

`hjkl` 連打ではなく、画面移動・検索・ジャンプを主に使う。

### 画面単位

| キー    | 動作                     |
| ------- | ------------------------ |
| `<C-u>` | 半ページ上               |
| `<C-d>` | 半ページ下               |
| `<C-f>` | 1ページ下                |
| `<C-b>` | 1ページ上                |
| `zz`    | カーソル位置を画面中央に |
| `zt`    | カーソル位置を画面上部に |
| `zb`    | カーソル位置を画面下部に |

### 行内移動

| キー      | 動作                           |
| --------- | ------------------------------ |
| `f{char}` | 行内で文字にジャンプ           |
| `F{char}` | 行内で逆方向に文字にジャンプ   |
| `t{char}` | 文字の手前にジャンプ           |
| `T{char}` | 逆方向で文字の手前にジャンプ   |
| `;`       | 直前の `f/t` を繰り返す        |
| `,`       | 直前の `f/t` を逆方向に繰り返す |
| `w` / `W` | 次の単語先頭へ                 |
| `b` / `B` | 前の単語先頭へ                 |
| `e` / `E` | 単語末尾へ                     |
| `0`       | 行頭へ                         |
| `^`       | 行頭（非空白）へ               |
| `$`       | 行末へ                         |

### ファイル内ジャンプ

| キー         | 動作                                      |
| ------------ | ----------------------------------------- |
| `/{pattern}` | 前方検索                                  |
| `?{pattern}` | 後方検索                                  |
| `n` / `N`    | 次/前の検索結果へ                         |
| `{数字}j`    | 相対行番号を見て下にジャンプ（例: `12j`） |
| `{数字}k`    | 相対行番号を見て上にジャンプ              |
| `{`          | 前の段落へ                                |
| `}`          | 次の段落へ                                |
| `gg`         | ファイル先頭へ                            |
| `G`          | ファイル末尾へ                            |
| `{数字}G`    | 指定行へ（例: `42G`）                     |
| `gd`         | 定義へジャンプ（LSP）                     |
| `%`          | 対応する括弧へ                            |

### 履歴ベース

| キー    | 動作                         |
| ------- | ---------------------------- |
| `<C-o>` | ジャンプリストで前に戻る     |
| `<C-i>` | ジャンプリストで次に進む     |
| `gi`    | 最後に Insert した位置に戻る |
| `g;`    | 変更リストで前の変更位置へ   |
| `g,`    | 変更リストで次の変更位置へ   |

### マーク

| キー         | 動作                           |
| ------------ | ------------------------------ |
| `m{a-z}`     | ローカルマークを設定           |
| `'{a-z}`     | マーク位置の行頭へジャンプ     |
| `` `{a-z} `` | マーク位置の正確な位置へジャンプ |
| `''`         | 直前のジャンプ位置に戻る       |

## Surround

### 基本操作

| 操作 | コマンド           | 説明                     |
| ---- | ------------------ | ------------------------ |
| 追加 | `ys{motion}{char}` | motion 範囲を char で囲む |
| 削除 | `ds{char}`         | char の囲みを削除        |
| 変更 | `cs{old}{new}`     | 囲みを old から new に変更 |

### 追加

| コマンド        | 前            | 後                          |
| --------------- | ------------- | --------------------------- |
| `ysiw"`         | `hello`       | `"hello"`                   |
| `ysiw'`         | `hello`       | `'hello'`                   |
| `ysiw)`         | `hello`       | `(hello)`                   |
| `ysiw(`         | `hello`       | `( hello )`                 |
| `ysiw]`         | `hello`       | `[hello]`                   |
| `ysiw[`         | `hello`       | `[ hello ]`                 |
| `ysiw}`         | `hello`       | `{hello}`                   |
| `ysiw{`         | `hello`       | `{ hello }`                 |
| `ysiw>`         | `hello`       | `<hello>`                   |
| `ysiw<div>`     | `hello`       | `<div>hello</div>`          |
| `ysiwt` + `div` | `hello`       | `<div>hello</div>`          |
| `ysiw`` `       | `hello`       | `` `hello` ``               |
| `yss"`          | `hello world` | `"hello world"`             |

### 削除 / 変更

| コマンド   | 前                 | 後                 |
| ---------- | ------------------ | ------------------ |
| `ds"`      | `"hello"`          | `hello`            |
| `ds)`      | `(hello)`          | `hello`            |
| `dst`      | `<div>hello</div>` | `hello`            |
| `cs"'`     | `"hello"`          | `'hello'`          |
| `cs")`     | `"hello"`          | `(hello)`          |
| `cst"`     | `<div>hello</div>` | `"hello"`          |
| `cs"t` + `span` | `"hello"`      | `<span>hello</span>` |

### ビジュアルモード

| コマンド   | 説明                           |
| ---------- | ------------------------------ |
| `S{char}`  | 選択範囲を char で囲む         |
| `gS{char}` | 選択範囲を char で囲む（改行付き） |

### motion / alias

| 項目  | 対象                    |
| ----- | ----------------------- |
| `iw`  | 単語                    |
| `aw`  | 単語 + 空白             |
| `ip`  | 段落                    |
| `it`  | タグの中身              |
| `at`  | タグを含む              |
| `b`   | `()`                    |
| `B`   | `{}`                    |
| `r`   | `[]`                    |
| `q`   | 近いクォート            |

## Treesitter

### Text Objects

| コマンド | 説明               |
| -------- | ------------------ |
| `vaf`    | 関数全体を選択     |
| `vif`    | 関数の中身を選択   |
| `daf`    | 関数全体を削除     |
| `cif`    | 関数の中身を変更   |
| `yaf`    | 関数全体をヤンク   |
| `vac`    | クラス全体を選択   |
| `vic`    | クラスの中身を選択 |
| `dac`    | クラス全体を削除   |
| `cic`    | クラスの中身を変更 |
| `vaa`    | 引数全体を選択     |
| `via`    | 引数の中身を選択   |
| `daa`    | 引数を削除         |
| `cia`    | 引数を変更         |

### 移動

| コマンド | 説明               |
| -------- | ------------------ |
| `]f`     | 次の関数の先頭へ   |
| `[f`     | 前の関数の先頭へ   |
| `]F`     | 次の関数の末尾へ   |
| `[F`     | 前の関数の末尾へ   |
| `]c`     | 次のクラスの先頭へ |
| `[c`     | 前のクラスの先頭へ |
| `]C`     | 次のクラスの末尾へ |
| `[C`     | 前のクラスの末尾へ |
| `]a`     | 次の引数へ         |
| `[a`     | 前の引数へ         |

### 覚え方

- `a` = around
- `i` = inside
- `f` = function
- `c` = class
- `]` = 次へ
- `[` = 前へ
