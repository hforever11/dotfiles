# Custom Keymaps

Leader キー: `<Space>`

## ESC 代替

| キー         | 動作                                                                      |
| ------------ | ------------------------------------------------------------------------- |
| `jk`         | ESC（インサート/コマンドライン/ビジュアル/セレクト/オペレータ待機/ターミナル） |
| `<Esc><Esc>` | ターミナルモードを抜ける                                                  |

ターミナルモードの `jk` は lazygit 内では無効（`k` がそのまま送られる）。

## インサートモード

| キー         | 動作                   |
| ------------ | ---------------------- |
| `<C-g><C-u>` | 単語を大文字に変換     |
| `<C-g><C-l>` | 単語を小文字に変換     |
| `<C-g><C-k>` | 単語の先頭文字を大文字に変換 |
| `<C-g><C-t>` | 単語の大小文字をトグル |

## ノーマルモード

### 検索

| キー         | 動作                                        |
| ------------ | ------------------------------------------- |
| `/`          | very magic 前方検索（正規表現が簡潔に）     |
| `?`          | リテラル後方検索（very nomagic）            |
| `n` / `N`    | 次/前の一致へ（カーソルを画面中央に保つ）   |
| `<Esc>`      | 検索ハイライトをクリア                      |
| `<leader>nh` | 検索ハイライトをクリア                      |

`/` と `?` はビジュアルモードでも同じ挙動。

### スクロール・行結合

| キー              | 動作                                   |
| ----------------- | -------------------------------------- |
| `<C-d>` / `<C-u>` | 半ページ移動後カーソルを画面中央に保つ |
| `J`               | 行結合してもカーソル位置を維持         |

### 削除（レジスタを汚さない）

| キー        | 動作                                      |
| ----------- | ----------------------------------------- |
| `x`         | 削除オペレータ（`"_d` 相当）              |
| `X`         | 行末まで削除（`"_D` 相当）                |
| `xx`        | 1行削除                                   |
| `x{motion}` | モーションで削除（例: `x3w` で3単語削除） |

### Undo / Redo（undo-glow）

| キー | 動作                   |
| ---- | ---------------------- |
| `u`  | Undo（ハイライト付き） |
| `U`  | Redo（ハイライト付き） |

### ヤンク / ペースト（yanky + undo-glow）

| キー              | 動作                                             |
| ----------------- | ------------------------------------------------ |
| `y`               | ヤンク（ヤンクリングに記録）                     |
| `p`               | カーソル後にペースト（ハイライト付き）           |
| `P`               | カーソル前にペースト（ハイライト付き）           |
| `<C-p>` / `<C-n>` | ペースト直後にヤンク履歴を前/後の項目へ差し替え |

ヤンク履歴の差し替えはペースト直後のみ有効（カーソルを動かすと解除）。

### 空行での自動インデント

| キー | 動作                                             |
| ---- | ------------------------------------------------ |
| `i`  | 空行なら自動インデント、そうでなければ通常の `i` |
| `A`  | 空行なら自動インデント、そうでなければ通常の `A` |

### 増減・トグル（dial.nvim）

カーソル下の数値・日付・真偽値・semver・論理演算子などを文脈判定で増減/トグルする。

| キー                          | モード | 動作                               |
| ----------------------------- | ------ | ---------------------------------- |
| `<C-a>` / `<C-x>`             | normal | カーソル下を増減 / トグル          |
| `<leader>+` / `<leader>-`     | normal | 同上（別名）                       |
| `<C-a>` / `<C-x>`             | visual | 選択内の数値をまとめて増減         |
| `g<C-a>` / `g<C-x>`           | visual | 選択行を 1,2,3… の連番に（階段状） |

対象例: `42` `0xff` `v1.2.9` `2024-01-31` `true/false` `&&`/`||` `==`/`!=` `and`/`or`

### 括弧ジャンプ（vim-matchup）

| キー | 動作                              |
| ---- | --------------------------------- |
| `M`  | 対応する括弧/キーワードにジャンプ |

`if`/`else`/`endif` などのキーワード間もジャンプ可能。カーソル下の単語が `end` を含む場合は `%`、そうでなければ `g%` で端から端へ直接ジャンプ。

### マクロ（q プレフィックス）

| キー          | 動作                               |
| ------------- | ---------------------------------- |
| `qq`          | マクロ記録開始（q レジスタ）       |
| `q`（記録中） | マクロ記録終了                     |
| `Q`           | マクロ再生（`@q`）                 |
| `qo`          | 他のウィンドウを閉じる（`:only!`） |
| `qt`          | 前のバッファに切り替え（`<C-^>`）  |

### パスコピー

| キー         | 動作                       |
| ------------ | -------------------------- |
| `<leader>cp` | 相対パスをコピー           |
| `<leader>cP` | 絶対パスをコピー           |
| `<leader>cn` | ファイル名をコピー         |
| `<leader>cN` | 拡張子なしファイル名をコピー |

### ウィンドウ操作

| キー         | 動作                     |
| ------------ | ------------------------ |
| `<leader>wv` | ウィンドウを縦分割       |
| `<leader>wh` | ウィンドウを横分割       |
| `<leader>we` | ウィンドウサイズを均等に |
| `<leader>wx` | 現在のウィンドウを閉じる |

### ウィンドウ / tmux ペイン移動（vim-tmux-navigator）

| キー                | 動作                                                   |
| ------------------- | ------------------------------------------------------ |
| `<C-h>` / `<C-j>` / `<C-k>` / `<C-l>` | 左/下/上/右へ移動（Vim ウィンドウがなければ tmux ペインへ） |

ターミナルモードでも有効。

### タブ操作

| キー         | 動作                             |
| ------------ | -------------------------------- |
| `<leader>to` | 新しいタブを開く                 |
| `<leader>tx` | 現在のタブを閉じる               |
| `<leader>tn` | 次のタブへ                       |
| `<leader>tp` | 前のタブへ                       |
| `<leader>tf` | 現在のバッファを新しいタブで開く |

## ビジュアルモード

| キー      | 動作                                             |
| --------- | ------------------------------------------------ |
| `x`       | 選択範囲を削除（レジスタを汚さない）             |
| `p` / `P` | ペースト（レジスタを維持、ハイライト付き）       |
| `<`       | インデントを減らして再選択                       |
| `>`       | インデントを増やして再選択                       |

## テキストオブジェクト

| キー                   | 動作                                     |
| ---------------------- | ---------------------------------------- |
| `i<Space>`             | WORD 内部（`iW` 相当）                   |
| `a"` / `a'` / `` a` `` | クォート内部（`2i"` 相当、クォート含む） |

## undo-glow ハイライト色

`config.core.theme` のパレット（現在は Catppuccin Latte）から取得。

| 操作   | 色                 |
| ------ | ------------------ |
| undo   | red (`#d20f39`)    |
| redo   | green (`#40a02b`)  |
| yank   | yellow (`#df8e1d`) |
| paste  | blue (`#1e66f5`)   |
| search | mauve (`#8839ef`)  |
